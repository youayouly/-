初始化
git branch
git checkout -b main

git init
git remote add origin https://github.com/youayouly/storage

查看和转换
git branch
git checkout main_1

修改文件内容 不增加文件
git add path/to/modified/file.txt
git commit -m "Update file content"
git push origin your_branch_name


删除后只要中间这里就行
增加
git pull origin main
git add .
git commit -m "Your commit message"
git push origin main

删除
git rm example.txt 任意操作
git add .
git commit -m "Remove example.txt"
git push origin main

拉取远程仓库 覆盖本地文件
git fetch origin
git reset --hard origin/main_1

main_1推送到main
先全部更新
git pull origin main
git fetch origin
git checkout main
git merge main_1 

如果出现合并错误
git merge --allow-unrelated-histories main_1
手动更改
git commit -m "Merge main_1 into main"  
git push origin main
